Application of XHTML coding rules

By: Anthony Miguel T. Ferrer

Hello! This HTML webpage uses the specific requirements for it to work on XHTML. Such as:

  1. Proper Nesting
  2. Proper tag closing; even empty tags need to be closed with a self-closing tag.
  3. Elements and Attributes need to be lowercased.
  4. A one-root element, that being the html tag.
  5. Values need to be properly quoted.
  6. Forbidden Attribute Minimization: A type of attributes should be a value equal to its attribute name.
  7. We use "amp instead of & in the URL's.
  8. Contents are within Block-Level Elements
